# 📚 Blog-App-Rod

The Blog app is a classic blog website. A fully functional website that will show the list of posts and empower readers to interact with them by adding comments and liking posts.

**Learning Objectives:**
- Understand how to use medium-fidelity wireframes to create a UI.
- Diagram a data model clearly and explain what it's doing.
- Practicing encapsulation using instance variables and methods.
- Running a Ruby program from the command line.
